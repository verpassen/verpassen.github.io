切屑已經有不同的研究，針對不同的外型，做命名分類

例如:
(誰、哪一年提出的，參考文章)

  • Flow type (流れ型)
  • Shera type(せん断型)
  • Tear type(むしい型)
  • Crack type(き裂型)

(誰、哪一年提出的，參考文章)
type i : Discontinuous chip
type ii : Continuous chip
type iii : Continuous chip with Build-up edge
type iV: Inhomogeneous strain-chip

同樣是切削，不同的材料特性、切削刀具的幾何設計、加工條件(切削速度，切削進給和切深)
讓最後得到的切屑外觀特徵截然不同
