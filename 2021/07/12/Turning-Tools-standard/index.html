4. Clamp Method 固定方式

書上所找到的固定方法有四種。 C、P、M、S。
各家刀具商會有自己的標稱，例如像Ceratizit 有D的標稱

Symbol Description Illustration
C 從刀片上方壓固
P 從中心楔型固定
M 刀片中心鎖固+上方壓固
S 中心鎖固螺絲

8. Hand of Tool 左/右手刀

有些型錄會寫著 "進給方向"
內孔刀具的方向，分成左手與右手
分辨方式，
以右手持刀桿，切削刃方向朝自己
裝刀片的卡槽在左手邊，為右手刀(Right-handed Tool)

以左手持刀桿，切削刃方向朝自己
裝刀片的卡槽在右手邊，為左手刀(Left-handed Tool)
