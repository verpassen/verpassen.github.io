Taylor Tool Life Equation

$$ vT^n = C $$

v: 切削速度(m/min)
T: 刀具壽命(min)
n & C: 常數
依據切削的進給、切深、工件和刀具材料等因素有所不同

切屑已經有不同的研究，針對不同的外型，做命名分類

例如:
(誰、哪一年提出的，參考文章)

  • Flow Type (流れ型)
  • Shera Type(せん断型)
  • Tear Type(むしい型)
  • Crack Type(き裂型)

(誰、哪一年提出的，參考文章)

  • type 1: Discontinuous chip
  • type 2: Continuous chip
  • type 3: Continuous chip with Build-up edge
  • type 4: Inhomogeneous strain-chip

同樣是切削，不同的材料特性、切削刀具的幾何設計、加工條件(切削速度，切削進給和切深)
